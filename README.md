# Ovyra Docs

Central product and operating documentation for Ovyra. This repository is organized to be useful for founders, designers, engineers, growth contributors, and AI agents working across the product.

## Included Documents

- `PRODUCT.md` product definition and user value
- `ROADMAP.md` phased execution plan
- `BRAND.md` visual and narrative system
- `ARCHITECTURE.md` platform architecture and system boundaries
- `PROMPTS.md` AI prompt strategy and guardrails
- `CONTENT_STRATEGY.md` editorial and growth plan

## Documentation Principles

- concise enough for fast decision-making
- specific enough to drive execution
- structured for both human collaborators and AI agents
